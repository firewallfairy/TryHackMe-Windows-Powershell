# PowerShell Basics

## Navigation
- `Get-Location` (`pwd`)
- `Set-Location C:\` (`cd`)
- `Get-ChildItem` (`ls`, `dir`)
- `Get-ChildItem -Force` (show hidden)

## Variables
- `$name = "Light"`
- `"$name is learning PowerShell"`

## Common cmdlets
- `Get-Date`
- `Get-ComputerInfo`
- `Get-Host`
- `Clear-Host` (`cls`)
